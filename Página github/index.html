Git init

El git init es para inicializar un repositorio de la carpeta en la cual estas posicionado en visual studio, desde la terminal y Git bash.

Git add

El git add agrega todos los archivos de la carpeta en la cual estas posicionado, al repositorio creado recientemente.

Git status

Git status es para ver el estatus en el que se encuentra mi repositorio local.

Git Commit

Git commit es para hacer una descripción de nuestros cambios

Git push

Sirve para guardar los cambios e insertarlos en el repositorio en Github
